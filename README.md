# truffle-contract-typescript

[![Join the chat at https://gitter.im/knockout/knockout](https://badges.gitter.im/Join%20Chat.svg)](https://gitter.im/truffle-contract-typescript?utm_source=badge&utm_medium=badge&utm_campaign=pr-badge&utm_content=badge)

Transform your truffle contract definitions into usable typescript classes

`npm install truffle-contract-typescript -g`

inside your project

`truffle-contract-typescript build ./path/to/contracts contracts.ts`
